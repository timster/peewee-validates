# Peewee Validates

A simple and flexible model validator for [Peewee ORM](http://docs.peewee-orm.com/en/stable/).

## Requirements

- Python 3 or later
- Peewee 2.8.0 or later

## Installation

This package can be installed using pip:

    pip install peewee-validates
